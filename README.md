# Christmas Lottery Tester

Using the the "El País" API to check your Lottery numbers.

https://servicios.elpais.com/sorteos/loteria-navidad/api/

## How to use

```shell
cp numbers.sample.js numbers.js
```

Fill your numbers ...

Remember:

```shell
npm i
```

And then:

```shell
npm start
```
